# Spring Boot Learning

## Summary

This repository contains projects and experiments created while learning backend development with Java and Spring Boot. It serves as a workspace for exploring core Spring Boot concepts, building REST APIs, and understanding backend architecture and development practices.

## Tech Stack

* Java
* Spring Boot
* Spring Web
* Spring Data JPA
* Maven

## Purpose

The purpose of this repository is to practice and improve backend development skills using Spring Boot. It will include various projects and implementations that demonstrate concepts such as REST API design, layered architecture, and general backend application development.
